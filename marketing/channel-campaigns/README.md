# Prevu channel campaign pack

This folder turns the skill-hub work into broader marketing assets for Prevu.

## Positioning

Prevu is the preview environment layer for coding agents.

Use this line consistently:

> Prevu gives coding agents VM-backed preview environments with public HTTPS URLs, SSH, services, and logs so humans can review running software before production.

Avoid calling Prevu "just staging URLs". The core category is preview environments for coding-agent work.

## Primary CTAs

- Try the CLI: `npm install -g @prevu/cli`
- Install the skills: `npm install -D @prevu/skills`
- Open source repo: https://github.com/prevu-cloud/prevu
- Website: https://prevu.cloud
- Discord: https://discord.gg/2a2MzpSX

## Channel priority

1. Hacker News / Reddit developer launch post.
2. X and LinkedIn founder/dev-tool thread.
3. Dev.to / Medium version of the long-form article.
4. Direct community replies in Claude Code, Codex, Cursor, OpenHands, OpenClaw, and AI-agent groups.
5. Newsletter and creator outreach with the skill-hub links.

## Files

- `launch-posts.md`: HN, Reddit, Dev.to/Medium, and Product Hunt launch copy.
- `social-posts.md`: X, LinkedIn, Discord, and Slack copy.
- `technical-article.md`: long-form article draft.
- `comment-reply-bank.md`: replies for common objections and launch-thread questions.
- `community-execution-plan.md`: account-safe publishing sequence for the first week.
- `directory-candidate-radar.md`: next directory targets ranked by fit and account risk.
- `directory-followups.md`: support messages for failed directory submissions.
- `email-submission-drafts.md`: ready-to-send directory emails for an approved brand mailbox.
- `outreach-list.md`: communities, creator notes, and cadence.
- `no-login-promotion-playbook.md`: promotion steps that avoid GitHub/OAuth authorization and personal-account posting.
- `publish-ready-article.md`: Dev.to/Medium-ready article with image slots and CTAs.
- `submission-field-bank.md`: reusable directory/form fields for future submissions.
- `visual-briefs.md`: social card, blog hero, diagram, and GIF directions.
- `assets/prevu-social-card.svg`: reusable social card.
