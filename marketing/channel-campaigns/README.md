# Prevu channel campaign pack

This folder turns the skill-hub work into broader marketing assets for Prevu.

## Positioning

Prevu is the preview environment layer for coding agents.

Use this line consistently:

> Prevu gives coding agents VM-backed preview environments with public HTTPS URLs, SSH, services, and logs so humans can review running software before production.

Avoid calling Prevu "just staging URLs". The core category is preview environments for coding-agent work.
